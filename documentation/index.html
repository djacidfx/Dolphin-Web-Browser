How to change the package name?

You can change it easily from Android Studio. Here are the steps:

  • 1.1 Open Android Studio
  • 1.2 Open the project using one of the following methods, (see Figure 1, 2)

Figure 1

Figure 2

  • 1.3 In your project pane, click on the little gear icon
  • 1.4 Uncheck / De-select the Compact Empty Middle Packages option (see Figure 3)

Figure 3

  • 1.5 Your package directory will be broken into individual directories now. Individually select each directory you want to rename
  • 1.6 Right-click on it
  • 1.7 Select Refactor and Click on Rename (see Figure 4)

Figure 4

  • 1.8 In the Pop-up dialog, click on Rename Package
  • 1.9 Enter the new name and hit Refactor
  • 1.10 Click Do Refactor at the bottom and allow a minute to let Android Studio update all changes
  • 1.11 Note: While renaming com in Android Studio, it might give you a warning. In such case, select Rename All.
  • 1.12 Open your AndroidManifest.xml file. Change the package to your new package name under the manifest tag. (see Figure 5)

Figure 5

  • 1.13 Now open your build.gradle (app) file. Update the applicationId to your new package name and sync gradle. (see Figure 6)

Figure 6

How To Create and Change Admob ID?

  • 1.24 If your app is in play store then select YES otherwise NO (see Figure 11)

Figure 12

  • 1.26 Then you will get an App ID. Save it at values > string.xml file. You will need it later. (see Figure 13)

Figure 13

  • 1.27 Change App Id in strings.xml file. (see Figure 14)

Figure 14

  • 1.28 Then click on CREATE AD UNIT and you will find three ad types. Only Banner and Interstitial types will be used here. (see Figure 15)

Figure 15

  • 1.29 Select Banner type. Provide Ad unit name and click on CREATE AD UNIT. (see Figure 16)

  • 1.30 Copy Banner Ad unit id and save it .You will need it later. (see Figure 17)

  • 1.31 Then again go back to Ad types and select Interstitial Ad type. Provide Ad unit name and select CREATE AD UNIT. (see Figure 18)
